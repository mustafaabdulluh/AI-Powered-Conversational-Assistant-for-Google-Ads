# AI-Powered Conversational Assistant For Google-Ads

Built an AI-Powered Conversational Assistant for Google Ads as a Retrieval-Augmented Generation (RAG) pipeline to provide expert-level support for Google Ads. The pipeline processes a 5,000-document bilingual (Arabic & English) knowledge base to deliver context-aware, accurate answers to complex advertising queries.

## Key Technical Contributions:

- **Bilingual RAG Pipeline**: Developed a custom RAG architecture using Python and LangChain that effectively handles cross-lingual retrieval, ensuring high accuracy for both Arabic and English queries.

- **Vector Search Optimization**: Implemented Chroma DB for efficient similarity searches, utilizing Hugging Face embeddings to map semantic relationships across a large-scale knowledge base.

- **Provider Agnostic LLM Layer**: Built a flexible backend with FastAPI that supports dynamic switching between OpenAI GPT and Google Gemini, allowing for performance benchmarking and cost optimization.

- **Data Engineering**: Curated and pre-processed a 5K-entry knowledge base, implementing advanced chunking strategies to maintain context and improve retrieval precision.

## Tech Stack:

Python, LangChain, FastAPI, Chroma DB, Hugging Face, OpenAI API, Google Gemini, Docker.
